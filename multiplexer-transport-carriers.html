Technical Specifications

IP Input Profile

MPEG 2 RTP v2 (1-185Mbps), UDP (1-185Mbps), SRT (1-60Mbps)
Connection: GigE (100/1000BaseT), 2x RJ45

IP Output Profile

MPEG 2 RTP v2 (1-185Mbps), UDP (1-185Mbps), SRT (1-60Mbps)
Connection: GigE (100/1000BaseT), 2x RJ45

ASI Input (TMC0100-0701)

EN500083-9, DVB-ASI 185Mbps (SPTS and MPTS), 2x standard BNC (75 Ohm)

Table Generation
